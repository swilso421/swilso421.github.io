Helpful Links

This page contains links to other helpful resources outside of this site. Learning is fun in groups, but sometimes you need to work through your own questions. Clicking on the images will redirect you to their sites.

W3 Schools

The W3 Schools is a website dedicated to providing free tutorials and references for many of the major web languages. You can go here to get explanations for the many HTML tags and CSS attributes. They even have editors that allow you to test out code without developing it. This is my go-to reference for web programming help.

Stack Overflow

Stack Overflow is a forum for amateur and professional programmers alike. If you can't figure something out, this is a great place to see if someone else had the same question as you. Or maybe you have a new question and can get it answered by fellow programmers. Since this is a public forum, you can get many different responses and solutions as you pick the brains of others.

Scratch

Scratch is a graphical programming language that is very easy to learn and use. Its drag-and-drop style of coding makes it simple for anyone to quickly make programs. Scratch is very useful for testing some basic programming concepts before applying them to a more complex language. Also, everything on the site is open-source. Feel free to look at others programs and learn by example.
